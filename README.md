 Telecom Customer Churn & Sales Analytics 

## Project Overview
This project focuses on analyzing customer behavior and sales performance for a Telecom provider.
The goal is to identify patterns leading to Customer Churn and provide insights into revenue growth.
I used  SQL for data cleaning (ETL) and Power BI for interactive visualization.

# Tech Stack
* Database:
           SQL (Data Cleaning & Transformation)
* Visualization:
          Power BI Desktop
* Language:
          DAX (Data Analysis Expressions)

# Data Cleaning Process (SQL)
- Duplicate Removal:
          Used CTEs to delete duplicate transaction records.
- Handling Nulls:
          Cleaned missing categorical and numerical data points.
- Standardization:
           Formatted phone numbers and plan names for consistency.
- Outlier Detection:
           Filtered unrealistic call durations (records > 24 hours).
- Feature Engineering:
           Segmented users based on data consumption patterns.
